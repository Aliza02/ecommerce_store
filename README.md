# Flutter E-commerce App

A comprehensive Flutter e-commerce application with an offline-first strategy, shopping cart persistence, Firebase backend, and Stripe payment integration.

## 🎯 Objectives
This project was built with the following objectives:

1. Implement a robust offline-first strategy for a seamless user experience.
2. Create a secure and user-friendly checkout flow with Stripe integration.
3. Apply test-driven development with comprehensive unit tests for API components.
4. Ensure smooth navigation and state management using the BLoC pattern.
5. Showcase the integration of multiple third-party services (Firebase, Stripe, location services).

## 🚀 Technologies Used
- Flutter
- Firebase (Authentication, Firestore)
- BLoC State Management
- Shared Preferences for local storage
- Stripe Payment SDK
- Geolocator for location services
- FakeStore API

## 🌟 Key Features

### 📱 User Experience
- **Offline-First Architecture**: Browse products, manage cart, and prepare orders even without an internet connection.
- **Smooth Navigation**: Intuitive user flow with elegant transitions.
- **Product Catalog**: Integration with FakeStore API for diverse product listings.
- **Search & Filtering**: Find products quickly with robust search capabilities.
- **Guest User**: Guest User Implementation to allow user to explore without creating an account.
  

### 🛒 Shopping Experience
- **Shopping Cart Management**: Add, update, and remove items from the cart.
- **Shopping Cart Persistence**: Save products for later viewing.
- **Product Details**: Comprehensive product information with images, descriptions, and specifications.
- **Order History**: Track past purchases and order status.

### 💳 Payment & Checkout
- **Stripe Integration**: Secure online payment processing.
- **Order Summary**: Clear breakdown of products, prices, and totals.
- **Address Management**: Edit and select shipping addresses.

### 📍 Location Services
- **Live Location Tracking**: Real-time location tracking.
- **Address Autocomplete**: Easy address input with location suggestions.

### 🔧 Technical Implementation
- **BLoC Pattern**: Clean state management with proper separation of concerns.
- **Firebase Backend**: Authentication, database, and cloud functions.
- **Unit Testing**: Comprehensive test coverage for API components.
- **Clean Architecture**: Well-organized codebase with BLoC Pattern.
- **Local Storage**: Efficient data persistence for offline functionality.

## 📦 Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/flutter-ecommerce-app.git
   
   ```

2. Install dependencies:
   ```sh
   flutter pub get
   ```

3. Run the app:
   ```sh
   flutter run
   ```
